## 데이터 소스

- **고용24 오픈API** — 국민내일배움카드 훈련과정 (XML, 무료)
  - 목록: `https://www.work24.go.kr/cm/openApi/call/hr/callOpenApiSvcInfo310L01.do`
  - 필수 파라미터: `authKey, returnType, outType, pageNum, pageSize, srchTraStDt, srchTraEndDt, sort, sortCol`
  - 핵심 응답 필드: 수강비 `COURSE_MAN` · 실훈련비 `REAL_MAN` · 취업률 `EI_EMPL_RATE3/6` · 취업인원 `EI_EMPL_CNT3` · 만족도 `STDG_SCOR` · 등급 `GRADE` · 정원 `YARD_MAN` · NCS/지역 코드
  - 인증키: ⚠️ **기업회원 전용(사업자등록번호 필수)** — 사용자 실업급여 수급(~2026.11) 종료 후 사업자등록 → 기업회원 가입 → 활용신청(심사 ~5영업일, 일 1,000회). 그 전까지는 공개 검색 페이지 소량 수집 + 공공데이터포털 파일데이터로 임시 파이프라인 (`.env`의 `WORK24_API_KEY`는 11월 이후)
  - ⚠️ 실호출로 검증할 것: REAL_MAN 의미(자부담 vs 지원금), 취업률 필드 커버리지 비율, 전체 과정 수, 트래픽 한도
